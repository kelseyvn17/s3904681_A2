Artificial Intelligence and Machine

AI can be known as human brain. If people are able to do or understand something, AI application is installed in computer system and work as human. It is applied into amazing applications with different functions such as speech recognition or computer vision. These function support to recognize the face, word or objects. AI is one of a field in computer science. It is an intelligence which human program to support computer automate the human’s behaviors.

In addition to artificial intelligence, we have machine learning. It is one topic inside A.I. It allows computers to be able to improve itself based on training data or experiences. It also can anticipate and make a decision without specific programming.In other words, machine learning in A.I will support computer identification basically like humans such as listen, see, and understand language, math, programming. Also, it might help humans manipulate information which is called Big Data.

Face recognization through camera which is put in the airport and buildings.

Virtual assistant (Siri, Google assistant, Alexa) can be able to hear, understand, answer and support the owner.

Some applications in medical, using AI to diagnose diseases based on X-ray, or MRI.

Smart TV also is combined with A.I tecchnology to improve image quality and recognize voice.

Cloud computing

Cloud computing is a model which provide resources to users through internet. Users can access any resources on Cloud in anytime and anywhere with the Internet. When an organization applies Cloud computing in their project, it will give them some benefits such as saving on expenses, convenience and safety.

Digital infrastructure is built these days. Smart elevators and parking, drone cars, trains and subways, farms and power plants - all will become safer and better managed thanks to the ability to Cloud storage and data analysis.

Access speed faster: In a system of cloud computing, there are a lot of servers from different long distances, but they still contain the same back-end in a website. Cloud computing can improve access to our website. If an organization engages in some huge system like Google or Amazon, users everywhere also go into the website as fast as customers in that country.

Website stability: A website with high stability will have less error of being inaccessible. Cloud computing makes companies immune to physical factors that can damage physical servers. If there is a cyber attack or virus, the website may still be unavailable. Cloud computing will automatically shut down the sites. Because this action will prevent the spread of malicious code into other sites and services in the organization's system.

I cannot imagine losing all of my crucial documents. Since I use Google which consists of Google Docs, Drive, Sheets and so on, I guess the lost thing does not occur. Most importantly, I really love it that I can return to the action which I have done recently. For example, in Google Docs, they still store the old versions of my documents. So, I can recognize what I changed and compare it to the present version.
